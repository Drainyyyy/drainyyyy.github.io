minimalLog

A minimal log library that makes console logs more viewable

General

  • Maintained: Yes
  • Source: Public (See links)
  • Language: Java
  • Creator(s): Drainyyy

    • Description

    minimalLog is a java library that makes logging easier.

    It makes logs more viewable and adds more important information like timestamps etc.

    For more view the README.md on GitHub (See links).

    Functions

  • ColoredLog (Set colors yourself or use default colors), UncoloredLog
  • Toggle printed timestamp
  • Toggle between different prefixes (notif for notification, info for information, warn for warning and err for error)
  • All functions (if there will be added more) can be found on GitHub (See links).
