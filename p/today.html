Overview

manageStream is a stream manager how you may have noticed by the name lol that got some useful features to make a stream look better.

manageStream will create files with the sub count, follower count, currently playing Spotify track etc.

This text files can be read by most of the streaming softwares and you can use the outputted data to display it in your stream.

Also its aim is to make streaming live more comfortable with useful features like automatic starting of things you need for the stream.
