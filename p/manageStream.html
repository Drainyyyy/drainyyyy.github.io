manageStream

manageStream makes live streaming more comfortable.

General

  • Maintained: Yes
  • Source: Public (See links)
  • Language: Java
  • Creator(s): Drainyyy

    • Description

    manageStream comes with a bunch of features that might be pretty useful for streamers to make their stream look nicer and manage it better.

    Also there will be more (hopefully useful) features added soon!

    If you got any feature-requests post them on the issues page of the projects GitHub.

    Functions

  • Uptime, Countdown, Time etc. writing into text files that can be read by streaming software
  • Opening your dashboard and starting your software
  • Planned: Spotify; Follower-, Sub-, Spectatercount, etc.;
