ypc

your personal chat

General

  • Maintained: Yes
  • Source: Public (See links)
  • Language: Python
  • Creator(s): Drainyyy

    • Description

    ypc is a chat platform that everyone can download and set up for themselves.

    This means you need to have a server, where your ypc chat can run on.

    Only people with the server id can join and they need to register.

    A server admin can accept or reject new users before they can join the chat.
