ToxicBot

A multifunctional Discord bot.

General

  • Maintained: Yes
  • Source: Public (See links)
  • Language: Python
  • Creator(s): Drainyyy

    • Description

    ToxicBot is a Discord bot with many functions.

    There are some Moderation commands, but mostly information and fun commands.

    Features can be requested by using the t.bugreport [bug] command.

    Functions

  • Moderation: Common kick, ban & clear chat
  • Fun: coinflip, 8ball, passwordgen, etc.
  • Information: About users, servers, the bot, bot ping, avatar infos, ...

    • Note: All commands can be found by using the t.commands command.
